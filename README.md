## Project Features
The laboratory exercise focuses on manual implementation to handle:
* Task 1: Grayscale transformation and manual histogram frequency calculation.
* Task 2: Contrast measurement using Michelson and RMS formulas.
* Task 3: Histogram Equalization using Cumulative Distribution Function (CDF).
* Task 4: Integrated processing pipeline for Low, Normal, and High contrast images.

## How to Run
1. Ensure jes4py and matplotlib are installed in your environment.
2. Clone this repository.
3. Run the main script: Task4_MainPipeline.py.
4. Follow the console prompts to select the required image categories.
5. Review results in the console and pop-up visualization windows.
